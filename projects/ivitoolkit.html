Multi GUI Solution for In-vehicle Infotainment

As a software developer at Embitel in Bangalore, I worked on Multi GUI Solution for In-vehicle Infotainment using the C++ based Qt framework. Along with team members, I developed a designer toolkit which allows quick design and development of a wide range of applications supporting a variety of platforms like navigation systems, mobile interfaces, measuring systems, multimedia devices, etc. It aims to simplify and speedup the creation of customized multi-platform applications with a simple drag and drop interface. This toolkit contains a data simulator, an integrated javascript editor, and an application preview mode. It provides an extensive database of widgets with WYSIWYG customizability. It aids in the creation and deployment of high-detail, easy-to-read GUIs within a short timeframe.
